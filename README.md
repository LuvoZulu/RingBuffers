# RingBuffers


RingBuffers is a collection of circular data structures. The planned data structures are :

1. Circular Arrays
2. Circular Linked Lists
3. Queues


RingBuffers final form is a header-only project and I will only have all the build tools and all that to test the code to ensure quality, etc.

This project is a living specification and I could use some advise in terms of design and its application. Do reach out :)

## Planned Features

- Generic Types
- Dynamic vs Static Allocation
